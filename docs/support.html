AudioBar Support

AudioBar is a macOS menu bar app for source volume control and local system EQ.

Permissions

System audio capture is used for local EQ processing. Automation permission is used only when you operate supported Music, Spotify, or Safari controls.

Troubleshooting

  • If the EQ does not start, confirm macOS granted system audio capture permission and relaunch AudioBar.
  • If Music, Spotify, or Safari controls do not work, open System Settings and confirm Automation permission for AudioBar.
  • AudioBar requires macOS 14.2 or newer.
