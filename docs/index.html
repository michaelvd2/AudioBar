AudioBar

Control source volume, web audio, and system sound from your menu bar. AudioBar adds quick mute, per-source L/R balance, mono or stereo playback, track controls, Safari media control, and a compact 11-band EQ for routed Mac audio.

What It Does

  • Mixes active audio sources with per-source volume, quick mute, L/R balance, and mono/stereo controls.
  • Controls supported Music, Spotify, Safari, and Safari Web App media with play/pause, previous/next, and 15-second rewind controls.
  • Shows the live system stream meter, keeps recently audible sources visible, and restores hidden rows from the footer.
  • Applies a custom vertical 11-band EQ to routed Mac sound with preamp, presets, reset, accent fills, and a visible 0 dB baseline.
  • Targets the actually playing Safari tab where supported, while keeping unsupported controls visibly unavailable.
  • Shows permission state from the footer before Accessibility, Input Monitoring, Automation, or system audio prompts are needed.

Install Notes

  • Requires macOS 14.2 or newer.
  • When the notarized zip is available, download it, unzip it, and move AudioBar.app to /Applications.
  • Use the footer permission button to revisit required macOS permissions when a control needs them.
  • macOS may ask for system audio capture, Automation, Input Monitoring, or Accessibility permission depending on which controls you use.
